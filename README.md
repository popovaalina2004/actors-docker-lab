Это Spring Boot REST API для управления базой данных актеров. Приложение предоставляет:

Основные функции:
Управление профилями пользователей:
Создание новых профилей (логин/пароль)
Поиск профилей по логину
Управление актерами:
Добавление актеров с привязкой к профилям
Редактирование информации об актерах
Удаление актеров
Поиск актеров по имени
Получение всех актеров для профиля
