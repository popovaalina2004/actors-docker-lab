Это Spring Boot REST API для управления базой данных актеров.

Основные функции:

1. Управление профилями пользователей:
создание новых профилей (логин/пароль), поиск профилей по логину

2. Управление актерами:
добавление актеров с привязкой к профилям, редактирование информации об актерах, удаление актеров, поиск актеров по имени, получение всех актеров для профиля
